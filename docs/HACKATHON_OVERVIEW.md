# Meta PyTorch OpenEnv Hackathon x SST — Overview

## What Is This?

India's first OpenEnv AI Hackathon, hosted by **Meta** in collaboration with **Hugging Face** and **PyTorch Foundation**. Participants build reinforcement learning environments using the OpenEnv framework.

**Prize Pool: $30,000**

| Place | Prize |
|-------|-------|
| 1st | $10,000 |
| 2nd | $4,550 |
| 3rd | $10,000 |
| 4th–8th | $2,000 each |
| 9th–15th | $650 each |

Winners also receive an **interview opportunity with Meta and Hugging Face AI teams** plus exclusive finalist merch.

## Round 1 — The Task

> Build a complete, real-world OpenEnv environment that an AI agent can learn from through the standard `step()` / `reset()` / `state()` API.

You choose 1 of 4–5 problem statements and build an OpenEnv environment around it.
